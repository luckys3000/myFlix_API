Movie API Documentation

The myFlix application provides access to a database of movies, allowing movie enthusiasts the ability to search for and retrieve extensive information on films, directors, and genres.

Users have the option to register, create a profile, and manage their favorite movies list.

Return data about a genre (description) by name/title (e.g., "Thriller") /movies/genre/:name GET None A JSON object holding data about a specific genre containing a description of the movie genre. Example: 

                Genre: {
                  Name: "Drama",
                  Description:
                    "The drama genre features stories with high stakes and many conflicts.

Return data about a director (bio, birth year, death year) by name /movies/directors/:name GET None A JSON object holding data about a specific movie's director, containing the director's name, short bio, birth year, and death year. Example: 

                {
                  Name: "George P. Cosmatos",
                  Bio: 'George Pan Cosmatos was a Greek-Italian film director and screenwriter.

                  co-productions.',

                  Death: "April 19, 2005",

Allow new users to register /users POST A JSON object holding data about the user that is to be registered. Example: 

                {
                  "name": "Charlie",
                  "favoriteMovies":[],
                }
A JSON object holding data about the user that is added including a Universally Unique Identifier (UUID). Example: 

                {
                  "name": "Charlie",
                  "favoriteMovies": [],
                  "id": "ff61f25d-0e23-428a-a0e4-57895fbcd705"
                }
Allow users to update their user info (username) /users/:id PUT A JSON object holding updated data about an existing user. Example: 

              {
                id: 1,
                name: "Jeffrey (update)",
                favoriteMovies: [],
              },
A JSON object holding data about the user's updated data. Example: 

              {
                id: 1,
                name: "Jeffrey (update)",
                favoriteMovies: [],
              },
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later); /users/:id/:movieTitle POST A JSON object holding updated data about an existing user. Example: 

            {
              "id": 1
              "name": "Jeffrey",
              "favoriteMovies": "Tombstone (update)"
            },
A plain text string notifying the user that the movie has been added to their favorites. Example:
"Tombstone has been added to user 1's favorites."
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later). /users/:id/:movieTitle DELETE A JSON object holding updated data about an existing user. Example:

"Tombstone has been removed from user 1's favorites."

              {
